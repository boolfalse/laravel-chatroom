
# ChatRoom

### Requirements
* [PHP version >= 7.1.3](https://laravel.com/docs/5.7/installation)


### Installation (with additional commands)

> Configure .env

```shell
composer install
php artisan clearcaches
php artisan cleanuploads
php artisan storage:link
php artisan droptables
php artisan migrate
php artisan db:seed
```
